## 📄 概述

本项目是一个**解析非标准 DOCX 文档并自动修正为标准格式的系统**。使用 Python 进行文档结构分析，结合多线程高速处理和大语言模型（LLM）文章重写。

---

## 🎯 背景与目的

企业和教育机构每天产生大量文档，但格式不统一、排版错误、标点误用等问题普遍存在。手动修正耗时耗力。

本系统旨在：

- **自动解析**文档结构
- **统一格式**
- 通过 LLM **提升文章质量**
- **大幅减少**工作时间

## 🔍 主要功能

### 1. 文档结构解析

解析 DOCX 的 XML 结构，提取：文字・段落、标题层级（H1〜H6）、表格・列表、图片・图表

![修正前文档示例](/images/图片2.png)

> ▲ 待修正文档示例。左侧为原文上传与解析，右侧显示检测到的问题与修正选项。

### 2. 自动修正处理（多线程）

Python `threading` 模块并行执行多项修正：符号・空格统一、标点符号转换、标题层级识别、模板插入、重复内容检测

### 3. LLM 文章重写

大语言模型提升文档质量：自然度・可读性改善、专业术语正确使用、文体统一、冗余表达削减

## 🛠 使用技术

| 技术 | 用途 |
|------|------|
| **Python 3.11** | 主开发语言 |
| **python-docx** | DOCX 文件读写 |
| **lxml** | XML 结构解析 |
| **threading** | 多线程并行处理 |
| **OpenAI API** | LLM 文章重写 |
