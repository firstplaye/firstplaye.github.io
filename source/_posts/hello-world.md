欢迎来到我的 Hexo 博客！这是你的第一篇文章。快速了解你可以做什么：

## 快速开始

### 创建新文章

```bash
$ hexo new "我的新文章"
```

### 本地预览

```bash
$ hexo server
```

### 生成静态文件

```bash
$ hexo generate
```

### 部署到 GitHub Pages

```bash
$ hexo deploy
```

更多信息请查阅 [Hexo 官方文档](https://hexo.io/zh-cn/docs/)。
